# 🕵️ Quem é o Impostor?

Um jogo social de dedução e blefe para grupos de amigos, otimizado para dispositivos móveis. Inspirado em jogos como Spyfall, esta versão traz uma interface moderna, animações fluidas e uma temática de "Agentes Secretos".

## ✨ Funcionalidades
- **Categorias Dinâmicas**: Mais de 150 palavras secretas.
- **Modo Mobile-First**: Interface pensada para ser usada em um único celular que passa de mão em mão.
- **Identidade Visual Premium**: Tema escuro com neon, ícones flutuantes e transições suaves.
- **Animações**: Desenvolvido com Framer Motion para uma experiência tátil e lúdica.

## 🎮 Como Jogar?
1. **Configuração**: Escolha o número de jogadores (mínimo 3).
2. **Distribuição**: Cada jogador, um por um, recebe o celular para ler sua palavra de código.
   - Os **Agentes** receberão a localização ou objeto secreto.
   - O **Impostor** receberá o alerta: "VOCÊ É O IMPOSTOR".
3. **Interrogatório**: Após todos lerem, inicia-se o debate. Cada um diz uma única palavra sobre o tema.
4. **Votação**: O grupo discute quem teve o comportamento mais suspeito e tenta eliminar o impostor.

- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React Icons](https://lucide.dev/)

## 🛠️ Instalação e Execução
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Gere o build de produção:
   ```bash
   npm run build
   ```
